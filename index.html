How it goes onto your WordPress site

Quick Launch tier — a 60-second paste. Premium tier — a one-click plugin install. Either way, your team does it without us needing access to the WordPress admin.

Quick Launch (snippet)

  1. WordPress admin → Appearance → Widgets (or the theme's footer code area).
  2. Add a "Custom HTML" widget to the footer.
  3. Paste the snippet we send you.
  4. Save. The widget appears on every page within a minute.
<!-- realty-connect-widget v1 -->
<link rel="stylesheet" href="https://cdn.../rc-widget.css">
<div id="rc-widget"></div>
<script async src="https://cdn.../rc-widget.js"
        data-target="https://chat.realtyconnect.tech">
</script>

Premium (plugin)

  1. WordPress admin → Plugins → Add New → Upload Plugin.
  2. Pick the realty-connect-widget.zip we send you.
  3. Click Install → Activate.
  4. Go to Settings → Realty Connect Widget: pick your design (A/B/C/D), upload a banner image, set the headline. Save.
  5. Visit your homepage to confirm.

Plugin works on any WordPress 5.x / 6.x install. Tested with the most common Indian themes (Astra, OceanWP, GeneratePress, Avada).

Quick questions

Will the widget slow down our website?

No. The script is loaded asynchronously and the widget itself is < 12 KB total (CSS + JS). It will not affect your Google PageSpeed score.

Can we change the banner image whenever we want?

On the Premium tier, yes — straight from the WordPress admin, no developer needed. On Quick Launch we can also do a small monthly retainer for banner swaps if you'd prefer.

What happens on a slow mobile connection?

The widget is the last thing to load on the page. If the user's connection is too slow it gracefully fails — the rest of your site still works perfectly.

Can we hide the widget on certain pages?

Included on Premium. You can hide it on pages like checkout, login, the privacy policy, or anywhere you don't want it.

Who owns the code?

You do, perpetually. After we deliver, you can keep it, modify it, host it yourselves — no recurring licence fee.
