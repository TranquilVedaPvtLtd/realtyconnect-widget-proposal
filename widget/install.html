Putting the Realty Connect widget on your WordPress site

Two ways to install. Path A is one line of code (recommended — future updates roll out automatically). Path B pastes the whole widget inline if you prefer everything self-contained.

Path A — one-line embed (recommended)

1. Open the place in WordPress where site-wide code goes

WordPress admin → Appearance → Widgets → drag a Custom HTML block into the Footer.
(If your theme has a "Footer Scripts" or "Custom Code" box under Theme Options, that also works.)

2. Paste this snippet

The whole install — one script tag. 
<script src="https://tranquilvedapvtltd.github.io/realtyconnect-widget-proposal/widget/embed.js" defer></script>

3. (Optional) Customise the banner text or promo

Add this before the script tag and edit any field you want. Anything you skip falls back to the default.

Optional config block. 
<script>
  window.RC_WIDGET_CONFIG = {
    headline:    "2 BHK in Baner under ₹1 Cr? Ask me anything.",
    kicker:      "Ask our AI assistant",
    promoTag:    "NEW",                            // small corner tag ("" to hide)
    ctaLabel:    "Chat",
    brand:       "Realty Connect AI",
    bannerImage: "",                               // e.g. "https://your-site.com/promo.jpg"
    chatUrl:     "https://chat.realtyconnect.tech",
    openInNewTab: true
  };
</script>
<script src="https://tranquilvedapvtltd.github.io/realtyconnect-widget-proposal/widget/embed.js" defer></script>

4. Save the widget and open your homepage

The chat widget appears bottom-right on every page within a minute. Click it to make sure it opens chat.realtyconnect.tech.

When you want to swap the banner image, change bannerImage to a public URL of your promo image (recommended ~640×240, < 200 KB). Save the widget. Refresh.

How to check it worked

  1. Open your homepage in an incognito / private window (so caching doesn't lie to you).
  2. Look at the bottom-right: a small card with a banner and a "Chat" button should appear.
  3. Click the card — a new tab should open at chat.realtyconnect.tech.
  4. Click the small × in the corner — the card should collapse to a small bubble.
  5. Refresh the page — the bubble stays (it remembers).

Common questions

Will it slow my website down?

No. The script is loaded with defer — it doesn't block your page. The widget itself is about 12 KB.

Can I hide the widget on the checkout / privacy pages?

With Path A, your theme's "page-specific code" feature can skip it. With Path B (inline), only paste the snippet on the pages you want it on. (Smart hide rules are built into the Premium plugin tier — separate quote.)

What if the script URL ever goes down?

Your site keeps working — only the widget would not render (the page itself is untouched). You can switch to Path B as a fallback at any time.

Does it work on mobile?

Yes. The card auto-sizes to phone widths and the tap targets are 44 px+ (per WCAG 2.1).

Can I change the chat destination?

Yes — set chatUrl in the config block.
