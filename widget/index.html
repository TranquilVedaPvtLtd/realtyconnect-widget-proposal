Two ways to install

A — One-line embed (recommended). Paste a single <script> tag. Updates roll out automatically.
B — Inline snippet. Paste the whole widget code inline; self-contained.
Both paths are documented step-by-step at install.html. The summary below.

Path A — One-line embed (recommended)

  1. WordPress admin → Appearance → Widgets → drag a Custom HTML block to the Footer.
  2. Paste this single line:
One line. 
<script src="https://tranquilvedapvtltd.github.io/realtyconnect-widget-proposal/widget/embed.js" defer></script>

Want to change the headline, promo tag, or banner image? Add a small window.RC_WIDGET_CONFIG block before the script tag — full options at install.html.
